📘 What is Terraform?

Terraform is an open-source Infrastructure as Code (IaC) tool that allows you to define, provision, and manage cloud infrastructure using configuration files.

🔧 Why Use Terraform?

  • Automates infrastructure deployment
  • Declarative syntax (.tf files)
  • Supports multi-cloud: Azure, AWS, GCP, etc.
  • Manages lifecycle: create, update, delete resources
  • Enables GitOps with version control

🧱 Terraform Folder Structure (Azure Example)


/terraform
├── main.tf
├── variables.tf
├── outputs.tf
├── terraform.tfvars

📄 Sample main.tf for Azure VM


provider "azurerm" {
  features {}
}

resource "azurerm_resource_group" "rg" {
  name     = "rg-techzone"
  location = "East US"
}

resource "azurerm_virtual_network" "vnet" {
  name                = "vnet-tech"
  address_space       = ["10.0.0.0/16"]
  location            = azurerm_resource_group.rg.location
  resource_group_name = azurerm_resource_group.rg.name
}

💡 Common Terraform Commands

  • terraform init – Initialize directory
  • terraform plan – Preview execution
  • terraform apply – Apply infrastructure
  • terraform destroy – Remove all infrastructure
  • terraform fmt – Format files
