🚀 What is Docker?

Docker is a platform that allows developers to package applications and their dependencies into lightweight, portable containers.

🧱 Basic Docker Concepts

  • Docker Image: Read-only template used to create containers.
  • Docker Container: Running instance of an image.
  • Dockerfile: Script that contains instructions to build a Docker image.
  • Docker Hub: Cloud-based registry for Docker images.
  • Volume: Persistent storage mechanism for Docker containers.

📦 Common Docker Commands

  • docker ps – List running containers
  • docker images – Show available images
  • docker build -t myapp . – Build Docker image from Dockerfile
  • docker run -d -p 8080:80 myapp – Run container from image
  • docker exec -it container bash – Access running container shell
  • docker logs container – View container logs
  • docker stop/start/restart container – Manage container lifecycle
  • docker rm container – Remove container
  • docker rmi image – Remove image

🔧 Real-World DevOps Use Cases

  • Run Nginx/Apache as web servers inside containers
  • Deploy microservices architecture
  • Set up ELK stack (Elasticsearch, Logstash, Kibana)
  • Monitor apps using Prometheus + Grafana containers
  • Automated build/test/deploy pipeline using Jenkins in Docker
  • Run databases like MySQL, MongoDB in isolated containers
